## Installation

To get a local copy up and running, follow these steps:

1. **Clone the repository**

    ```bash
    git clone https://github.com/your-username/your-repo-name.git
    ```

2. **Navigate to the project directory**

    ```bash
    cd your-repo-name
    ```

3. **Install dependencies**

    ```bash
    npm install
    ```
